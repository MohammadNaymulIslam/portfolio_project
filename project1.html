Network Infrastructure Design & Segmentation

Project Goal

To build a secure, segmented, and manageable corporate network infrastructure simulating a real-world office environment, focusing on efficient routing, IP management, and access control.

Action & Technologies Used

  • Designed and configured multiple VLANs to logically separate departments (e.g., Finance, HR, IT).
  • Configured inter-VLAN routing using Cisco routers with both Static and OSPF protocols.
  • Set up DHCP servers for dynamic IP allocation and DNS for internal name resolution.
  • Applied Access Control Lists (ACLs) to routers to control traffic flow and limit access between VLANs.
  • Configured Cisco and MikroTik devices for routing, NAT, and firewall rules.

Outcome

Successfully simulated a segmented and policy-driven network, reinforcing knowledge of enterprise routing, VLAN implementation, and traffic control using ACLs. The project showcased the ability to design and secure networks based on business requirements.
