Virtual Lab Setup for Network & System Simulation

Project Goal

To create a virtualized lab environment that enables simulation of enterprise-level networking and systems administration for testing and learning purposes.

Action & Technologies Used

  • Built and configured topologies in EVE-NG, GNS3, and VMware Workstation.
  • Simulated networks using Cisco and MikroTik router images for realistic testing.
  • Integrated Linux and Windows VMs for hybrid environment simulation.
  • Performed lab exercises on routing, switching, DNS, DHCP, and firewall rules.

Outcome

Developed a complete virtual infrastructure for hands-on practice, increasing troubleshooting capabilities and system integration understanding without physical hardware dependency.
