Kubernetes Deployment & Management

Project Goal

To set up and manage a Kubernetes cluster for container orchestration, simulating real-world deployment pipelines for scalable and resilient application hosting.

Action & Technologies Used

  • Installed and configured a Kubernetes cluster with one master and multiple worker nodes.
  • Deployed Pods, Services, and ReplicaSets using YAML configurations.
  • Exposed applications using ClusterIP, NodePort, and LoadBalancer services.
  • Performed rolling updates and rollbacks on deployments.
  • Used kubectl for cluster monitoring and management.

Outcome

Acquired hands-on experience in managing containerized workloads in Kubernetes. Demonstrated the ability to deploy, scale, and maintain cloud-native applications in a production-like environment.
